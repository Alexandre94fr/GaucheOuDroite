DOCUMENTATION:
==============

If you want to modify the content of the files inside the Shared folder.
You should ALWAYS do it from the file inside the Shared folder.

I say that because the FrontEnd and the BackEnd will be able to see the files inside the Shared folder like if they are really inside the FrontEnd or BackEnd folder.
But those are copy, so modifying the copy won't impact the FrontEnd AND the BackEnd.


Note:
The copies are not manually copies, I didn't copy and paste the different files inside the different folders.
No, it's automatic, when you modify the files inside the Shared folder, the copies inside the FrontEnd and BackEnd folders will be updated automatically.

If you add new files or folders inside the Shared folder, be sure to manually update the FrontEnd files.
Unity doesn't catch that new files have been added inside the Shared folder.
